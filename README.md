## 1 INTRODUÇÃO
O Python começou a ser desenvolvido no final da década de 1980 pelo programador holandês Guido van Rossum, com o objetivo de criar uma linguagem de programação mais prática, simples e acessível em comparação às linguagens existentes na época. O projeto foi iniciado no CWI (Centro de Matemática e Ciência da Computação), na Holanda, e teve seu lançamento oficial em 1991.

Um dos principais diferenciais do Python é sua sintaxe clara e organizada, utilizando indentação para estruturar o código, o que facilita a leitura e compreensão por parte dos desenvolvedores. O nome da linguagem foi inspirado no grupo de comédia britânico Monty Python.

Atualmente, o Python é considerado uma das linguagens de programação mais populares do mundo, principalmente por ser fácil de aprender e utilizar. Além disso, possui compatibilidade com diferentes sistemas operacionais, conta com uma ampla variedade de bibliotecas prontas e pode ser aplicada em diversas áreas, como desenvolvimento web, automação de tarefas, análise de dados, ciência de dados e inteligência artificial.

## 2 DESENVOLVIMENTO

O Python pode ser utilizado em diversas áreas da tecnologia, sendo uma das linguagens mais versáteis da atualidade. Entre suas principais aplicações, destacam-se:

- Desenvolvimento de sites e aplicações web;
- Automação de tarefas e processos;
- Ciência de dados e análise de informações;
- Inteligência artificial e aprendizado de máquina;
- Desenvolvimento de jogos;
- Segurança da informação;
- Desenvolvimento de sistemas desktop;
- Computação científica;
- Criação de scripts e ferramentas de produtividade.

Sua ampla utilização no mercado de trabalho ocorre principalmente pela facilidade de integração com outras tecnologias e pela grande comunidade de desenvolvedores que contribuem constantemente com novas bibliotecas e soluções.

### 2.1 VANTAGENS E DESVANTAGENS

Entre as principais vantagens do Python, destacam-se:

- Sintaxe simples e fácil de aprender;
- Código organizado e de fácil leitura;
- Grande quantidade de bibliotecas prontas;
- Compatibilidade com diferentes sistemas operacionais;
- Alta produtividade no desenvolvimento;
- Comunidade ativa e ampla documentação.
- Por outro lado, a linguagem também apresenta algumas desvantagens, como:
- Menor desempenho em comparação com linguagens compiladas, como C e C++;
- Maior consumo de memória em determinados projetos;
- Não é a melhor opção para aplicações que exigem processamento extremamente rápido;
- Dependência de interpretador para execução dos códigos.
  
Mesmo com essas limitações, o Python continua sendo uma das linguagens mais utilizadas no mundo, principalmente devido à sua praticidade, flexibilidade e variedade de aplicações no mercado tecnológico.

### 2.2 EXEMPLOS DE CÓDIGO

**Print, Input e Printf**
Os comandos print e input são utilizados para interação básica com o usuário dentro do Python. O print é responsável por exibir informações na tela, enquanto o input permite receber dados digitados pelo usuário durante a execução do programa. Já o conceito de printf, muito comum em outras linguagens como C, no Python é realizado utilizando o próprio print, combinado com formatação de strings. Esses recursos são fundamentais para entrada e saída de dados em aplicações simples e complexas. 

**String, Inteiro e Booleano**
Os tipos de dados são utilizados para representar diferentes tipos de informações dentro de um programa. O tipo string armazena textos e caracteres, o tipo inteiro (int) representa números inteiros, e o tipo booleano (bool) trabalha com valores lógicos, sendo True para verdadeiro e False para falso. Esses tipos são essenciais para manipulação de dados e controle de funcionamento do sistema. 

**Array, Append e For**
Os arrays, conhecidos no Python como listas (list), são estruturas utilizadas para armazenar múltiplos valores em uma única variável. O método append permite adicionar novos elementos à lista de forma dinâmica. Já o laço for é utilizado para percorrer elementos de listas e executar repetições de maneira organizada. Esses recursos facilitam o tratamento de conjuntos de dados e automatizam tarefas repetitivas. 

**If e Else**
As estruturas condicionais if e else são utilizadas para tomada de decisões dentro do programa. O if verifica se uma condição é verdadeira, permitindo executar determinadas ações, enquanto o else define o que acontecerá caso a condição seja falsa. Essas estruturas tornam o programa mais inteligente e capaz de responder a diferentes situações.

**While**
O laço de repetição while é utilizado para executar um bloco de código repetidamente enquanto uma condição for verdadeira. Esse recurso é bastante útil em situações que exigem repetição contínua, como menus interativos, validação de dados e processos automáticos. É importante garantir que a condição seja alterada corretamente para evitar repetições infinitas.

**Def**
A palavra-chave def é utilizada para criar funções em Python. As funções permitem organizar o código em blocos reutilizáveis, facilitando a manutenção, leitura e reutilização do programa. Com elas, é possível executar tarefas específicas sempre que necessário, tornando o desenvolvimento mais estruturado e eficiente.

## 3. POUCO MAIS SOBRE 

### 3.1 EMPRESAS QUE UTILIZAM 

**Google**

De todas as grandes empresas que usam Python, o Google é talvez a maior entusiasta da linguagem no mundo. Junto com Java e C++, Python é uma das linguagens oficiais da gigante californiana. O Google adotou o Python quando a linguagem ainda era relativamente nova e não tão popular, defendendo seu uso com a já famosa frase: “Python where we can, C++ where we must”.  O primeiro crawler do Google — o robô responsável por fazer pesquisas e trazer os resultados corretos quando o usuário faz uma busca — foi feito em Python.  Hoje, o Google usa Python para coletar e analisar dados, além de desenvolver algoritmos de machine learning. A própria biblioteca TensorFlow, amplamente utilizada para aprendizado profundo, foi 
desenvolvida pelo Google e é escrita em Python. 

**Instagram**

Python é a principal linguagem de programação utilizada no Instagram. Mesmo com diversas mudanças ao longo do tempo em sua stack tecnológica, o aplicativo não teria nascido sem Python, pois foi construído sobre o Django, um framework web desenvolvido em Python.  Atualmente, com mais de um bilhão de usuários, os desenvolvedores do Instagram continuam criando verificadores estáticos de código em Python para investigação dos servidores, que possuem milhões de linhas escritas na linguagem. 

**Netflix**

A Netflix nunca escondeu que usa Python — e muito. De acordo com seus próprios desenvolvedores, a linguagem é usada no ciclo de vida completo de seus conteúdos, desde ferramentas de segurança até algoritmos de recomendação e redes de distribuição de conteúdo.  Embora os engenheiros da Netflix tenham autonomia para escolher a linguagem com que desejam trabalhar, a maioria prefere Python, atraída por sua biblioteca padrão, o rico catálogo de bibliotecas de terceiros, a simplicidade da linguagem e sua comunidade ativa de desenvolvimento.  A empresa também utiliza Python para automação de processos internos, como monitoramento de servidores e otimização na entrega de conteúdo. 

**Spotify**

Duas forças principais levaram o Spotify a escolher Python: a capacidade da linguagem de acelerar o desenvolvimento e a comunidade ativa que existe ao redor dela.  O Spotify faz amplo uso do Python para análise de dados e personalização de recomendações musicais, utilizando algoritmos baseados no comportamento de escuta dos usuários.  Os desenvolvedores da plataforma também usaram Python para construir a infraestrutura responsável por rodar as previsões de comportamento dos usuários. 

**Uber**

O Uber começou com duas linguagens principais: Node.js para o time de Marketplace e Python para todos os demais. Essas primeiras linguagens ainda alimentam a maioria dos serviços que rodam no Uber hoje.  A empresa usa Python para análise de dados e para criar algoritmos de machine learning que ajudam a melhorar a precisão das previsões de tempo de chegada dos motoristas e das estimativas de preços. 
Facebook (Meta)
Python é amplamente utilizado no Facebook no desenvolvimento de infraestrutura e na criação de ferramentas internas. A linguagem é valorizada pela sua facilidade de aprendizado, legibilidade e pela vasta biblioteca de recursos disponíveis. O Facebook também utiliza Python para análise de dados e machine learning, ajudando a melhorar a experiência dos usuários e a personalização dos conteúdos exibidos na plataforma.  Python é a terceira linguagem mais popular dentro do Facebook, logo atrás de C++ e Hack. 
Industrial Light & Magic (ILM)
Um exemplo menos óbvio, mas igualmente relevante, é o da Industrial Light & Magic, empresa de efeitos especiais fundada por George Lucas. O departamento de CGI da ILM foi fundado quatro anos após a criação da empresa e escolheu Python como linguagem de operação, motivado pela rapidez oferecida pela linguagem e sua facilidade de interoperação com outras linguagens. Dentro da empresa, Python é usado em várias frentes, como a manutenção de um banco de dados de todas as imagens já produzidas pelo estúdio.

### 3.2 CURIOSIDADES

**O nome não tem nada a ver com cobra!**

Muitas pessoas associam o nome à cobra, devido ao logotipo da linguagem. No entanto, a verdadeira inspiração por trás do nome é bem diferente.  Na década de 1970, a BBC tinha um programa de TV popular do qual Guido van Rossum era um grande fã, chamado “Monty Python’s Flying Circus”. Assim, quando desenvolveu a linguagem, ele pensou que precisava de um nome curto, único e um pouco misterioso, e decidiu chamar o projeto de “Python”.

**Existe um poema escondido dentro do Python**

Se você escrever import this no interpretador do Python e executar, será retornado um poema cujo título é “The Zen of Python”, escrito por Tim Peters sobre a linguagem de programação.  O poema traz princípios filosóficos que guiam o desenvolvimento em Python, como “Bonito é melhor que feio”, “Simples é melhor que complexo” e “A legibilidade conta”.
